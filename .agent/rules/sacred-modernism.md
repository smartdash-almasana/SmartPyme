# Sacred Modernism - Fábrica SmartPyme

Este documento establece el estándar de oro para el proceso de fabricación e implementación de SmartPyme.

## 1. Lógica Fiscal y Cálculos Monetarios
Toda la lógica de cálculo monetario o cálculo de impuestos (p. ej., IVA) en Python debe declarar el uso de `decimal.Decimal`. Bajo nunguna circunstancia se admitirá el uso de variables del tipo "floating-point" para valores transaccionales.

## 2. Estándares Estéticos (UI)
El Frontend o interfaz de usuario generada por los agentes deberá abrazar de forma inquebrantable el patrón tipográfico de **Manrope** y/o **Inter**. El objetivo central de estos esquemas es presentar información estructurada, amparados bajo los principios de visibilidad clara, pureza y estricta elegancia funcional.
